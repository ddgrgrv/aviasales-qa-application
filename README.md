# QA Engineer Application — Aviasales

Одностраничный продуктовый лендинг для отклика на вакансию QA Engineer в Aviasales.

## Стек

- React + TypeScript
- Tailwind CSS v4
- Framer Motion
- Vite

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```

## Деплой на GitHub Pages

1. Создайте репозиторий на GitHub и запушьте код
2. Установите зависимости: `npm install`
3. Задеплойте: `npm run deploy`

Сайт будет доступен по адресу `https://ddgrgrv.github.io/aviasales-qa-application/`
